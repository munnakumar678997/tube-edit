A personal, self-hosted fork of an open-source YouTube WebView client. Distributed under the MIT License (see LICENSE).

Click the button above to download the latest signed APK (built automatically from `main` by GitHub Actions).


## Features
* Google Gemini video summaries (customisable prompts and models)
* Video / Shorts downloader with built-in muxer
* Thumbnail and captions downloader
* Ad blocker
* Minimize video / Picture-in-Picture
* Gesture control for volume and brightness
* Dislike count display
* Background audio player
* Media codec toggles
* Sponsor segment skipping
* Force zoom, hide shorts, up to 10x playback speed

This is an educational project showcasing JavaScript injection into a WebView. It is not affiliated with or endorsed by YouTube/Google.
